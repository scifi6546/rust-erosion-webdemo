Shallow Water Equations

Written By Nicholas Alexeev

Shown below is a simulation of water. It is running entirely on the web browser using a self built engine. As described in the links above it uses the shallow water equations. Three different scenarios are offered for simulations. Click on the grey button under the name of a scenario to start a simulation. The view can be changed by left click and dragging the mouse.
